About Us

Our platform connects users facing vehicle issues in Sri Lanka with nearby mechanics through a mobile app. Real-time tracking facilitates quick assistance. Three user categories are involved, and both a mobile and web app streamline the process, aiming to enhance the connection between users and mechanics.

  • User Categories

    The system consists of 03 user categories. They are User, Garage Mechanic and the admin.

  • System Functionalities

    The Real-time Location Tracking, User Autonomy and Communication Features are some of the system Functionalities in our project.

  • Using Navigations

    This empowers users to make timely decisions, ensuring efficient assistance in critical moments on the road.
